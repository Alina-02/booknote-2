## Basic functionalities

- [x] Add, update and delete books.
- [x] Add, update and delete quotes.
- [] Sign in, sign up and sign out.
- [] Dark mode
- [] Search bar

## More functionalities

- [] Order the book's menu by genre.
- [] Books search bar
- [] Favourite books
- [] Favourtie quotes
